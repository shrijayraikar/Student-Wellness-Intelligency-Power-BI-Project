#Student Wellbeing Intelligence Dashboard
-Balance. Thrive. Succeed...

An interactive Power BI dashboard analyzing student survey data to uncover how social media usage relates to academic performance, mental health, sleep, and relationships.

Highlights:-
* 6 connected report pages: Executive Overview, Mental Health & Lifestyle, Academic Impact, Relationship & Conflicts, an Interactive Story View, and a Drill-through Student Profile page

* Custom DAX measures: average usage hours, % affected academically, mental health banding, addiction count, and conflict level

* Bookmark-driven toggle views (Academic vs. Gender lens) and drill-through to individual student profiles

* Built with Power BI Desktop, Power Query, and DAX on a two-table data model (Student Details + Platform Details)
